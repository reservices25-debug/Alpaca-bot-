# Alpaca-bot-

A simple trading bot using the Alpaca API for automated stock trading.

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Set up your Alpaca API credentials as environment variables:
   ```
   export APCA_API_KEY_ID='your_api_key'
   export APCA_API_SECRET_KEY='your_secret_key'
   ```

3. Run the bot:
   ```
   python bot.py
   ```

## Features

- Get account information
- Place market orders (buy/sell stocks)

## Disclaimer

This is a basic example. Use paper trading for testing. Real trading involves risk.
